Functions

Return to table of contents

All functions that can be called are listed below. Functions are called via thier alias
Main functions, as they are called in CHScript, are functions that are built for the engine but
are linked to the scripting language as well so they may be used through scripting.
The functions are listed exactly as they are meant to be called. If you don't spell
something right, or you use the wrong capitol/lowcases then the function wont be recognized.

INT STRING TEXT VAR N/A ? FILE
A number Text in quotes "" Text in brackets <> A variable Can be blank Depends on param 1 A filename

Function Param 1 Param 2 Param 3 Example Desciption

func Sub-Function ? ? func set_title "Hello world!" Sets the title of the window
write VAR or TEXT - - write [Hello again!] | write my_variable Outputs text to the window
break - - - break Adds a line break/starts a new line
load - - - load Loads (a) hard coded file(s)
save - - - save Saves (a) hard coded file(s)

log STRING VAR or blank INT log "Logging a success!" blank 1 | log "Logging a number: " my_number 1 Writes to the log file
call FILE - - call test.chs Opens, and runs a script
clear - - - clear Clears the window of all text
empty - - - empty Clears the log
dump - - - dump Alias of empty; Clears the log

Variables

Return to table of contents

There are only two variables in CHScript. More will be added in the future, however the
ones available are currrently: Integers and Strings. You create variables, set variables, and
use variables very similarly to C++. However, all variables are global and permanent. The
only way to remove a variable is via the "free" function, or it's alias "del". Unlike C++ you
can't create a variable, and set it's value at the same time you must do it seperate.
NOTE: It is best to remove uneeded variables; More variables means slower access time!

Creating, Setting, and Using Variables

#Creating a integer and a string
int my_int
string my_string

#Setting the values for my_int and my_string
int my_int = 3
string my_string = "Hello!"

#Increment my_int by 2
int my_int += 2
#Decrement my_int by 1
int my_int -= 1

#Ouput my_string
write var my_string
#Get some text input
inputText
#Assign the input text to my_string
string my_string = last_text
#Output my_string again
write var my_string

#Perform some math
#Add 2 to my_int
math my_int + 2
#Multiply the previous product by 2
math last_math * 2

#Assign the previous product to my_int
my_int = last_math

#Output my_int
write var my_int





Loops & Ifs

Return to table of contents

If statements are very simple, and run a script if they are correct. While loops too are very simple, and
keep running a script as long as the loop is correct. If and while loops use variables for checking, and
a filename for the script they run. There are also script loops, which is when a script calls itself
somewhere in it's code causing it to run itself repeatedly.

If statements

Return to table of contents

If statements compare two variables. These may be a integer that you created being compared to
a number you specify, two integers you created being compared, a integer and a randomly chosen
number, a random number and a random number, and so on. If statements compare numbers only
currently. If statements will be expanded upon in the future.

Symbol Usage Operation
> if x > 10 :: test.chs Greater than
< if x < 10 :: test.chs Less than
>= if x >= 10 :: test.chs Greater than or equal to
<= if x <= 10 :: test.chs Less than or equal to
= if x = 10 :: test.chs Equal to
!= if x != 10 :: test.chs Not equal to

If statement example

#Creating a integer
int my_int

#Setting the value of the integer to a random number
int my_int = rand

#Checking if the integer is less than, or equal to 50
#If it is, then call the script 'got_it.chs'
if my_int <= 50 :: got_it.chs

While Loops

Return to table of contents

While loops perform a script repeatedly as long as the comparison of two variables is true.
These may be a integer that you created being compared to a number you specify, two integers
you created being compared, a integer and a randomly chosen number, a random number
and a random number, and so on. While loops compare numbers only currently. The while
loop is basicly a if statement that repeats itself.

Symbol Usage Operation
> while x > 10 :: test.chs Greater than
< while x < 10 :: test.chs Less than
>= while x >= 10 :: test.chs Greater than or equal to
<= while x <= 10 :: test.chs Less than or equal to
= while x = 10 :: test.chs Equal to
!= while x != 10 :: test.chs Not equal to

While loop example

#Creating a integer
int my_int

#Setting the value of the integer
int my_int = 2

#While my_int is less then 50, repeat while_1.chs
while my_int <= 50 :: while_1.chs

#Then, inside while_1.chs we can put
int my_int += 1
#This means that everytime while_1.chs is called,
#we will increase my_int by 1. So, once my_int reaches
#51 it will stop looping

Script Loops

Return to table of contents

Script looping is very important if your making something like a game.
Script looping is where you have a script call itself (Generally at the
very end of the script) to cause it to loop itself forever. Script loops are best
used for a overarching loop that calls other scripts, and performs all the
main functions of your game. Such as for example you would have it call
a script that updates enemy monsters, then a script to draw text on the screen that
represents those monsters, then another script that handles input from the player,
and so on. Script looping can be used for as a while loop as well through the use
of script calls and if checks. While looping is a compact way of doing that though.

Script loop example

#Clear the screen
clear

#Write hello world!!!!!
write [Hello world!!!!!]

#Run the main script; The same one
#we are in, causing a loop.
call main.chs





Output

Return to table of contents

There are four output functions, they are: write, showInput, TextColor, and break
write will write variables, or text to the window. showInput will display the last text input that
was recieved via inputText. break will start a new line, and TextColor will set the color of
all output following it.

Output example

#Create a integer
int my_int
#Set the value of my_int to a random number
int my_int = rand

#Output 'Hello World!'
write [Hello world!]
#New line
break

#More writing
write [The value of my_int is ]
#Set the text color to 2 (Green)
TextColor = 2
#Output my_int
write var my_int
break
#Pause until we recieve any key press
pause

Math

Return to table of contents

There is a single master math function. When you call the function you specify two variables and a operation.
These two variables may be intergers (int), numbers you choose, rand, last_math, or last_number.
The operation is either adding ( + ), subtracting ( - ), multiplying ( * ), or dividing ( / ). You can also do
incrementing or decrementing of integers when you set thier value, you simply use "-=" or "+=" instead of
a equal ( = ) sign followed by the amount to increment or decrement. A example script can be found below;
the example script shows all four operations and incrementing & decrementing.

Math example

#Create int a
int a
#Set the value of int a to a random number
int a = rand

#Display int a
write var a
break

#Adding 2
math a + 2
#Set the value of a to the last math problem we did
int a = last_math

#Subtracting 1
math a - 1
#Set the value of a to the last math problem we did
int a = last_math

#Multiplication by 3
math a * 3
#Set the value of a to the last math problem we did
int a = last_math

#Division by 2
math a / 2
#Set the value of a to the last math problem we did
int a = last_math

#Display int a after all the math is done
write [After adding 2, subtracting 1, multiplying by 3, then dividing by 2 I got: ]
write var a
break

#Increment int a by 1
int a += 1
#Decrement int a by 3
int a -= 3

write [After incrementing by 1, then decrementing by 3 I got: ]
write var a
break
